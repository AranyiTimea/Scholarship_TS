Task: Create a generic class with a generic array and following methods:
• void Add(T element) - Adds the given element to the end of the list
• T Remove(int index) - Removes the element at the given index
• bool Contains(T element) - Prints if the list contains the given element (True or False) • void Swap(int index1, int index2) - Swaps the elements at the given indexes • int CountGreaterThan(T element) - Counts the elements that are greater than the given element and prints their count
• T Max() - Prints the maximum element in the list
• T Min() - Prints the minimum element in the list
